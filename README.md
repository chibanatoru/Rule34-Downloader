# Rule34 Downloader with API Key

A Rule34 downloader using an API key.

## Features

- Download media using the Rule34 API
- Convert downloaded files into CBZ format
- Custom request timeout support

## Credits

This project is based on:

## API Key

To generate an API key, visit:

https://rule34.xxx/index.php?page=account&s=options

A Rule34 account is required to generate an API key.
